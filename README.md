🔫 INSAS Rifle VR Training Simulator
📌 Overview

A Virtual Reality (VR) based INSAS rifle training simulator developed using Unity.
This project allows users to dismantle, assemble, and perform firing practice in an immersive and interactive environment using VR controllers.

⚡ Developed بالكامل (end-to-end) by a single developer in 35 days

🚀 Features
🔧 Dismantling & Assembling

🔧 Dismantling Module

The dismantling module allows users to interactively disassemble the INSAS rifle in a step-by-step process using VR controllers.

Magazine
Rear Coil Spring
Gas Piston Tube
